MAGICSSIAN

What is it

Magicssian is a CSS Boilerplate to quickstart a small web project with pure css trickery and a bit of design.

Get into it

Get the latest version
Download
Add the following tag to your html5 document 

<link rel="stylesheet" href="./path/to/magicssian.min.css">
and Tadaa! you can start doing some tricks.
